I created basic kinematic player controller. Unity version that i used for the project is Unity 2018.4 LTS. There are 2 classes. "PhysicsObject" and "PlayerPlatformerController". The PlayerPlatformerController class inherits from PhysicsObject

PlayerPlatformer takes inputs and moves the player.

PhysicsObject controls the physics system. It checks for collisions by casting vertical and horizontal rays.
